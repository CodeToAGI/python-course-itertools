# CodeToAGI Episode 35 - Itertools Challenge

**Student × Subjects Schedule Planner**

Solution to the challenge presented in **Episode 35: Python itertools Deep Dive**.

## What This Project Demonstrates

- `product()` → Generate all student-subject combinations
- `combinations()` → Create possible 2-student study groups
- `groupby()` → Group data by subject (after sorting!)
- `cycle()` → Assign round-robin time slots
- `islice()` → Limit output (Bonus)
